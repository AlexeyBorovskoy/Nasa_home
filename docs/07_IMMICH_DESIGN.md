# 07. Immich

## 1. Роль в проекте

Immich используется как фото- и видеоархив, близкий по пользовательскому сценарию к Google Photos.

## 2. Ограничение для Jetson Nano

Официальные требования Immich указывают минимум 6 GB RAM и рекомендацию 8 GB, а для систем с 4 GB RAM допускается работа с отключёнными ML-функциями. Поэтому на Jetson Nano 4 GB необходимо начинать с отключённого machine learning.

## 3. Рекомендуемый режим

```text
IMMICH_DISABLE_MACHINE_LEARNING=true
видеотранскодирование: выключено или минимально
массовый импорт: не выполнять до нагрузочных тестов
```

## 4. Данные

```text
/mnt/storage/immich/library
/mnt/storage/db/immich-postgres
```

## 5. Тест первого запуска

1. Создать пользователя.
2. Подключить Android Immich.
3. Загрузить 20–50 фото.
4. Загрузить 2–3 видео.
5. Проверить `docker stats`.
6. Проверить температуру Jetson.
7. Проверить размер thumbnails/transcoded data.

## 6. Риски

| Риск | Мера |
|---|---|
| Нехватка RAM | отключить ML, увеличить swap, ограничить контейнеры |
| Нагрузка CPU | отключить тяжёлые фоновые задачи |
| Рост storage | учитывать +10–20% на миниатюры и производные файлы |
| БД на HDD | проверить стабильность USB и SMART |
