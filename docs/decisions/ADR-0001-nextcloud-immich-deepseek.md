# ADR-0001: Выбор Nextcloud + Immich + DeepSeek Gateway

## Статус

Принято.

## Контекст

Необходимо создать домашнее семейное облако на Jetson Nano + USB HDD.

## Решение

Использовать:

- Nextcloud для файлов, документов, контактов и календаря;
- Immich для фото и видео;
- DeepSeek API через LLM Gateway для административных LLM-функций;
- не использовать локальную LLM на первом этапе.

## Последствия

Плюсы:

- готовые Android-клиенты;
- понятная архитектура;
- низкая нагрузка по AI на Jetson;
- возможность публикации проекта;
- расширяемость до собственного Android-клиента.

Минусы:

- внешняя LLM требует privacy-фильтра;
- Immich на 4 GB RAM требует отключения ML;
- Nextcloud требует аккуратной настройки БД/Redis/cron;
- для настоящего backup нужен второй носитель.
