# Миграция с Google — NASA Home Cloud

Пошаговая замена Google-сервисов самохостингом.
Всё делается постепенно — Google не отключается до подтверждения, что данные перенесены.

---

## Обзор миграции

```
Google Photos ──────────────────────→ Immich
Google Drive  ──────────────────────→ Nextcloud Files
Google Контакты ────────────────────→ Nextcloud Contacts (CardDAV)
Google Календарь ───────────────────→ Nextcloud Calendar (CalDAV)
Gmail ──────────────────────────────→ не мигрируем (Stage 5)
Google Maps offline ────────────────→ не мигрируем
```

---

## Фаза 0. Экспорт из Google (Google Takeout)

> Делается ДО настройки приложений. Без экспорта нет данных для импорта.

### 0.1 Скачать Google Takeout

1. Открыть [takeout.google.com](https://takeout.google.com)
2. **Снять выбор со всего** → выбрать только нужное:
   - ✅ **Google Фото** → Все фото альбомы
   - ✅ **Контакты** → формат vCard (.vcf)
   - ✅ **Календарь** → формат iCal (.ics)
   - ✅ **Диск** → все файлы (опционально, большой объём)
3. Формат: zip, максимальный размер архива 50 GB
4. Нажать **Создать экспорт** → придёт email со ссылкой (от нескольких часов до 2 дней)
5. Скачать архивы

### 0.2 Структура Takeout

```
Takeout/
├── Google Фото/
│   ├── Фото со мной/
│   ├── Photos from 2023/   (каждый год отдельно)
│   │   ├── IMG_20230115.jpg
│   │   ├── IMG_20230115.jpg.json  ← метаданные (дата, GPS)
│   │   └── ...
│   └── ...
├── Контакты/
│   └── Контакты.vcf        ← все контакты в одном файле
└── Календарь/
    ├── Личный.ics
    └── Праздники в России.ics
```

## Фаза 1. Миграция фотографий → Immich

### Метод A: Через веб-интерфейс (небольшие коллекции, < 5 GB)

1. Открыть `http://192.168.0.50:2283` (или HTTPS-адрес)
2. Войти в Immich
3. **+** → **Загрузить** → перетащить папки из Takeout
4. Immich читает метаданные из `.json` файлов Google (дата, GPS)

### Метод B: Через CLI (большие коллекции, рекомендуется)

Immich включает инструмент `immich-go` для массового импорта с сохранением метаданных.

```bash
# На машине с архивом Takeout (Windows/Linux):

# 1. Скачать immich-go
# https://github.com/simulot/immich-go/releases
# Выбрать версию для Windows: immich-go_Windows_x86_64.zip

# 2. Запустить импорт
./immich-go upload \
  --server http://192.168.0.50:2283 \
  --api-key YOUR_IMMICH_API_KEY \
  --google-photos \
  "E:/Takeout/Google Фото/"
```

**Получить API ключ Immich:**
- Immich Web → Аватар → Настройки аккаунта → API ключи → Новый ключ

### Что происходит при импорте

- Immich читает `.json` метаданные Google → восстанавливает **оригинальные даты** фото
- Дубликаты определяются по хешу файла — повторно не загружаются
- GPS-координаты сохраняются (карта в Immich работает)
- Live Photos (MOV + JPG) объединяются автоматически

### Проверка импорта

```
Immich → Explore → По дате — хронология должна восстановиться
Immich → Карта — GPS-точки должны появиться
```

## Фаза 2. Миграция контактов → Nextcloud Contacts

### 2.1 Импорт в Nextcloud

1. Открыть `http://192.168.0.50:8080` → Nextcloud
2. **Контакты** (значок в верхнем меню)
3. ⚙️ → **Импортировать** → выбрать `Контакты.vcf` из Takeout
4. Дождаться импорта

### 2.2 Настроить синхронизацию на телефоне

После установки DAVx⁵ ([ANDROID_SETUP.md → Шаг 4](ANDROID_SETUP.md)):

1. DAVx⁵ синхронизирует Nextcloud Contacts → стандартное приложение Контакты Xiaomi
2. Контакты появятся в телефоне через 1–2 минуты после первой синхронизации

### 2.3 Отключить Google Contacts sync

После проверки (все контакты на месте):

Настройки → Аккаунты → Google → выключить **Контакты**

> ⚠️ Не удалять аккаунт Google целиком — только синхронизацию контактов.

---

## Фаза 3. Миграция календарей → Nextcloud Calendar

### 3.1 Импорт в Nextcloud

1. Nextcloud → **Календарь**
2. ← (левое меню) → **Импорт календаря** → выбрать `.ics` файл из Takeout
3. Повторить для каждого календаря (Личный, Праздники и т.д.)

### 3.2 Синхронизация на телефоне

DAVx⁵ автоматически найдёт новые календари после синхронизации.
Проверить: DAVx⁵ → Аккаунт NASA → CalDAV → список календарей ✅

### 3.3 Отключить Google Calendar sync

Настройки → Аккаунты → Google → выключить **Календарь**

---

## Фаза 4. Миграция файлов → Nextcloud

### 4.1 Загрузить через веб

1. Nextcloud Web → **Файлы** → **Загрузить** → выбрать папки из `Takeout/Диск/`
2. Для больших объёмов использовать WebDAV-клиент (Cyberduck, RaiDrive)

### 4.2 WebDAV адрес

```
http://192.168.0.50:8080/remote.php/dav/files/USERNAME/
или
https://nasa.YOUR-DOMAIN.com/remote.php/dav/files/USERNAME/
```

### 4.3 Настроить авто-синхронизацию папок

Nextcloud Android → **⋮** → **Автозагрузка**:
- `Documents` → `Documents/`
- `Download` → `Downloads/` (по желанию)

---

## Фаза 5. Отключение Google Photos

После проверки что ВСЕ фото в Immich (сравнить количество):

1. Google Фото → ⚙️ → Управление хранилищем → убедиться что есть резервные копии
2. Отключить **резервное копирование Google Фото** на телефоне:
   Google Фото → Профиль → Настройки Фото → Резервное копирование → Выкл.
3. Через месяц (после проверки) — очистить Google Photos
4. Immich → включить авто-загрузку новых фото с телефона
