# NASA Home Cloud — New Phone Setup Guide

> 🇷🇺 Полный гайд по подключению Android-телефона к NASA Home Cloud.  
> Шаги выполняются один раз при смене телефона или первом подключении.

> 🇬🇧 Complete guide for connecting an Android phone to NASA Home Cloud.  
> Steps performed once when changing phone or connecting for the first time.

---

## Шаг 0 / Step 0 — Что вам нужно / What you need

| Параметр | Значение |
|---|---|
| Сервер (внешний) | `193.8.215.130` |
| Nextcloud HTTPS | `https://193.8.215.130:8443` |
| Immich HTTP | `http://193.8.215.130:2283` |
| Immich LAN (дома) | `http://192.168.0.50:2283` |
| Ваш логин | см. памятку в `artifacts/users/` |
| Домашний WiFi | TP-Link_828C |

---

## Шаг 1 / Step 1 — Immich (фотографии / photos)

**Скачать:** Play Store → **"Immich"**

**Настройка:**
1. Открыть → адрес сервера: `http://193.8.215.130:2283`
2. Email: `<login>@nasa.local`
3. Пароль: из памятки
4. Включить **резервное копирование**
5. Настройки → Сеть → добавить локальный WiFi URL: `http://192.168.0.50:2283`
   (выбрать сеть: TP-Link_828C)

**MIUI/HyperOS (Xiaomi):** дополнительно выполнить:
- Настройки → Приложения → Immich → Батарея → **Нет ограничений**
- Настройки → Приложения → Immich → **Автозапуск → Вкл**
- Разрешить работу в фоне в диспетчере задач (закрепить в RAM)

---

## Шаг 2 / Step 2 — Nextcloud (файлы / files)

**Скачать:** Play Store → **"Nextcloud"**

**Настройка:**
1. Открыть → Войти
2. Адрес сервера: `https://193.8.215.130:8443`
3. При предупреждении о сертификате → **Подтвердить**
4. Логин / пароль: из памятки

**MIUI/HyperOS:** Батарея → Нет ограничений; Автозапуск → Вкл

---

## Шаг 3 / Step 3 — DAVx⁵ (контакты и календарь / contacts & calendar)

**Скачать:** [davx5.com](https://www.davx5.com/download) или F-Droid → **"DAVx⁵"**

> ℹ️ В Play Store платная версия — с F-Droid или сайта бесплатно.

**Настройка:**
1. Открыть → нажать **"+"** (добавить аккаунт)
2. Выбрать **"Войти с URL и именем пользователя"**
3. URL: `https://193.8.215.130:8443/remote.php/dav`
4. Логин / пароль: из памятки
5. Подтвердить сертификат
6. Отметить галочками: **Contacts ✓  Calendar ✓**
7. Нажать **↻ Синхронизировать**

**Проверка:** открыть стандартные **Контакты** Android → убедиться что виден аккаунт DAVx⁵

---

## Шаг 4 / Step 4 — Nextcloud Contacts & Calendar (опционально)

**Скачать из Play Store:**
- **"Nextcloud Contacts"** — редактирование контактов в браузере-стиле
- **"Nextcloud Calendar"** — просмотр/создание событий

Эти приложения работают поверх DAVx⁵ и дают более удобный интерфейс.

> Браузер покажет предупреждение о сертификате → **Дополнительно → Перейти**

## Что делать если что-то не работает / Troubleshooting

| Проблема | Решение |
|---|---|
| Immich не подключается | Проверить подключение к интернету; убедиться что порт 2283 открыт |
| DAVx⁵ "ошибка сертификата" | Нажать "Принять" при первом подключении |
| Контакты не синхронизируются | DAVx⁵ → аккаунт → кнопка ↻; проверить что Contacts включён |
| Immich не делает бэкап автоматически | MIUI: Батарея → Нет ограничений + Автозапуск |
| Забыл пароль | Написать Алексею — он сбросит через admin |

---

## Восстановление при смене телефона / Restore on new phone

При смене телефона все данные восстанавливаются автоматически:

| Данные | Источник | Действие |
|---|---|---|
| Фотографии | Immich сервер | Войти → все фото появятся |
| Контакты | Nextcloud CardDAV | Настроить DAVx⁵ → синхронизация |
| Календарь | Nextcloud CalDAV | Настроить DAVx⁵ → синхронизация |
| Файлы | Nextcloud Files | Войти → все файлы доступны |

> 🚀 Никаких резервных копий на новый телефон не нужно — всё хранится на сервере.
