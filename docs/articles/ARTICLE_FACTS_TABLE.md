## Несоответствия — приоритет исправления

### 1. Число фото/видео (КРИТИЧНО для читателей)
Три числа в статье:
- **6 697** — «загружено с телефона» (Шаг 5, «Что получилось»)
- **6 719** — caption к `android_immich_backup_stats.jpg` ("6697 из 6719 загружено")  
- **6 484 фото + 210 видео = 6 694** — данные API (Шаг 7, таблица)

**Объяснение несоответствия:**
- 6 719 = всего медиафайлов в галерее телефона (включая системные/скрытые)
- 6 697 = сколько Immich принял к загрузке (очередь)
- 6 694 = сколько Immich фактически хранит (после дедупликации)
- Разница 6 697 − 6 694 = 3 файла — вероятно, повреждённые или дублированные файлы

**Рекомендация:** Добавить одно предложение с объяснением: «Immich не принял 3 файла (повреждённые/дубли) — это норма».

### 2. RAM 2.3 GB — это snapshot из примера Telegram-отчёта
Реальное потребление может отличаться. Не выдавать за точное актуальное значение.
