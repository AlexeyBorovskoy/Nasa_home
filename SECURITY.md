# Security Policy

## Reporting

Open a private security advisory or contact the repository owner.

## Scope

This project is a blueprint. Users must adapt it to their environment.

## Sensitive data

Never submit:

- real `.env` files;
- API keys;
- photos/videos;
- database dumps;
- SSH keys;
- personal backup manifests.

## LLM privacy

DeepSeek API is used only through LLM Gateway and only for anonymized diagnostics and project documentation in Stage 1.
